* npm i - установка необходимых для работы модулей
  (если не написать, то ниодна из команд работать не будет).

* npm run dev - запуск облегчённого режима.

* npm run build - запуск полного режима.

* npm run zip - создание архива папки dist.

* npm run svgSprive - создание svg спрайта.

* npm run deploy - выгрузка папки dist на удалённый сервер.
  (для работы нужно ввести настройки в файл gulp\config\ftp.js).

* перед началом написания кода скачай и добавь все нужные шрифты в src\fonts.

* также скачай все иконки, добавь их в src\svgicons после пропиши npm run svgSprive.

* компилируются все файлы *.pug которые находятся непосредственно в папке src\
  (вложенные папки не обрабатываются).

* компилируется только src\scss\style.sass.

* всё что тут было изначально не трогать. 

* папка dist создастся сама

* Файл fonts.sаss создастся сам при добовлении шрифтов.

* для обновления файла fonts.sаss его нежно удалить и запустить любой из режимов.
